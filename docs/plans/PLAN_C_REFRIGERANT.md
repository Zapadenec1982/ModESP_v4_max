# Блок C: Таблиці насичення фреону — ДЕТАЛЬНИЙ ПЛАН

> **Статус: DESIGNING (85%)**
> Досліджено: data sources (NIST/CoolProp), point density (50 pts = ±0.3°C), Antoine не підходить для blends.
> Незалежний від інших блоків. Чисто обчислювальний код, найлегший для тестування.

---

## Контекст

Superheat = T_evap_outlet − T_saturation(P_suction). Для розрахунку T_sat з тиску потрібні таблиці насичення фреону. Antoine equation НЕ працює для zeotropic blends (R404A, R448A, R449A) — тільки lookup tables.

## Ключові рішення (з дослідження)

- **Lookup table > Antoine equation** — R404A/R448A/R449A = zeotropic blends, Antoine coefficients не існують
- **CoolProp** для генерації таблиць — Python, REFPROP-accurate, всі 5 фреонів підтримуються
- **50 точок/фреон** з 2°C кроком → ±0.3°C interpolation error (краще за ±0.5°C target)
- **Flash:** ~300B/refrigerant × 5 = **1.5KB total** (constexpr → DROM, 0 RAM)
- **Runtime:** binary search + lerp = **<100us** per lookup

## Покрокова реалізація

**C1: CoolProp table generator**

`tools/generate_refrigerant_tables.py`:
```python
import CoolProp.CoolProp as CP

REFRIGERANTS = {
    "R404A": {"range": (-45, 65), "step": 2},
    "R134a": {"range": (-45, 65), "step": 2},
    "R290":  {"range": (-45, 65), "step": 2},
    "R448A": {"range": (-45, 65), "step": 2},
    "R449A": {"range": (-45, 65), "step": 2},
}

for name, cfg in REFRIGERANTS.items():
    for T_C in range(cfg["range"][0], cfg["range"][1]+1, cfg["step"]):
        T_K = T_C + 273.15
        P_Pa = CP.PropsSI('P', 'T', T_K, 'Q', 1, name)  # Q=1 = dew point (for suction)
        P_bar = P_Pa / 1e5
        # emit as C++ constexpr
```

Output: `components/modesp_refrigerant/generated/saturation_tables.h`:
```cpp

struct SatPoint { int16_t t_c10; uint16_t p_mbar; };
// t_c10 = temperature × 10 (e.g., -125 = -12.5°C)
// p_mbar = pressure in millibar (e.g., 3500 = 3.5 bar)

static constexpr SatPoint R404A_TABLE[] = {
    {-450, 93},    // -45.0°C, 0.093 bar
    {-430, 115},   // -43.0°C, 0.115 bar
    // ... ~55 points ...
    {650, 3850},   // +65.0°C, 38.50 bar
```

**Формат:** `int16_t` + `uint16_t` = 4 bytes/point (vs 8 bytes for two floats). Compact.

**C2: Interpolation API**

`components/modesp_refrigerant/include/modesp/refrigerant/saturation.h`:
```cpp
enum class Refrigerant : uint8_t {
    R404A = 0, R134a, R290, R448A, R449A, COUNT
};

// Returns saturation temperature in °C for given pressure in bar
// Returns NAN if pressure out of table range
float saturation_temp(Refrigerant ref, float pressure_bar);

// Returns saturation pressure in bar for given temperature in °C
float saturation_pressure(Refrigerant ref, float temp_c);

// Get refrigerant name string
const char* refrigerant_name(Refrigerant ref);
```

`saturation.cpp`:
```cpp
float saturation_temp(Refrigerant ref, float pressure_bar) {
    const auto& table = get_table(ref);  // returns span<const SatPoint>
    uint16_t p_mbar = (uint16_t)(pressure_bar * 1000.0f);

    // Binary search for bracket
    size_t lo = 0, hi = table.size() - 1;
    while (lo < hi - 1) {
        size_t mid = (lo + hi) / 2;
        if (table[mid].p_mbar <= p_mbar) lo = mid;
        else hi = mid;
    }

    // Linear interpolation
    float alpha = (float)(p_mbar - table[lo].p_mbar)
                / (float)(table[hi].p_mbar - table[lo].p_mbar);
    return (table[lo].t_c10 + alpha * (table[hi].t_c10 - table[lo].t_c10)) / 10.0f;
}
```

**C4: Host tests (doctest)**

```cpp
TEST_CASE("R404A saturation at known points") {
    // NIST reference: R404A at 1.0 bar → ~-46.5°C (dew)
    float t = saturation_temp(Refrigerant::R404A, 1.0f);
    CHECK(t == doctest::Approx(-46.5f).epsilon(0.5f));  // ±0.5°C

    // R404A at 10.0 bar → ~23.5°C
    t = saturation_temp(Refrigerant::R404A, 10.0f);
    CHECK(t == doctest::Approx(23.5f).epsilon(0.5f));
}

TEST_CASE("Out of range returns NAN") {
    CHECK(std::isnan(saturation_temp(Refrigerant::R404A, -5.0f)));  // below min
    CHECK(std::isnan(saturation_temp(Refrigerant::R404A, 50.0f)));  // above max
}

TEST_CASE("All refrigerants have valid tables") {
    for (int r = 0; r < (int)Refrigerant::COUNT; ++r) {
        float t = saturation_temp((Refrigerant)r, 5.0f);
        CHECK(!std::isnan(t));
        CHECK(t > -50.0f);
        CHECK(t < 50.0f);
    }
}
```
